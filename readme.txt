== Description ==
Clean Coupons is a plugin to clean up expired coupons from Woocommerce. This allows you to free up space and unclutter your coupon list.

- Remove expired coupons from Woocommerce
- Schedule cleanup hourly, daily or twice daily
- Have option to either trash it or permanently delete it

== Installation ==
Unzip and Upload Folder to the /wp-content/plugins/ directory.
Activate through WordPress plugin dashboard page.
Visit TheCartPress > Coupon Cleanup to change your settings.
Save Changes.

== Frequently Asked Questions ==
=Why are the expired coupons still not removed?=
The cleanup is done according to the interval chosen in settings. To prevent heavy load on the server, each cleaning session will remove 10 coupons.
